    suitability.py:
    - COVERED_STATES set contains exactly 21 state names (all ALLCAPS with & for J&K, Andaman)
    - is_deficient(profile, nutrient) → True when profile[nutrient]["low"] > 50
    - score_crop(profile, tolerances) → higher score for crops whose min_tolerance <= block's actual category; N-tolerant crop (N_min="low") scores high even when block has 96% low-N
    - rank_crops(profile, crop_rows) → returns list sorted descending by score, max 5 results, only crops with score > 0

    fertiliser.py:
    - generate_fertiliser_advice(profile, threshold=50) → returns list of advice dicts
    - When Nitrogen low_pct=96 > 50 → returns dict with nutrient="Nitrogen", low_pct=96, message starts with "96%", fertiliser_recommendation contains "Urea"
    - When Phosphorus low_pct=40 <= 50 → no advice card for Phosphorus returned
    - "Potential Of Hydrogen" (pH) NOT included in fertiliser advice (pH is a range check, not deficiency)
    - Empty profile → empty list returned
    - All threshold boundary: low_pct=50 → no advice (threshold is strict >)
    - low_pct=51 → advice generated

    seeding (tested via integration — see Task 2):
    - parse_pct("97%") → 97, parse_pct("0%") → 0, parse_pct(" 97% ") → 97
    - Malformed % value raises ValueError caught by try/except, row skipped, counter incremented
